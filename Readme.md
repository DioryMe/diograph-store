# Template: Webpack & Typescript & Jasmine

## Install dependencies

```
npm install
```

## Compile

```
npm run compile
```

Compiles Typescript into Javascript starting from `app/index.ts`.
Compiled Javascript is placed into `lib/`.

## Tests

```
npm test
```

Jasmine tests are placed in `spec/` folder.
